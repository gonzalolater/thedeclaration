# Seguridad

## Política de Seguridad

Este documento describe las prácticas de seguridad para el proyecto "The Declaration".

### Reportar Vulnerabilidades

Si descubres una vulnerabilidad de seguridad, por favor:

1. **NO** la reportes en los issues públicos
2. Contacta al equipo de seguridad de forma privada
3. Proporciona detalles sobre la vulnerabilidad y pasos para reproducirla

### Estándares de Seguridad

- Mantener todas las dependencias actualizadas
- Revisar el código antes de hacer merge
- Usar variables de entorno para datos sensibles
- Validar todas las entradas de usuario
- Usar HTTPS para todas las conexiones

### Dependencias

Las dependencias se mantienen actualizadas regularmente. Reporta cualquier vulnerabilidad conocida.

### Contacto

Para consultas de seguridad, contacta al equipo del proyecto.
